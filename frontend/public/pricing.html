常见问题

免费版和付费版有什么区别？

免费版每天50次分析，单次不超过10,000字符。专业版无限分析、支持100,000字符以上代码、提供CLI/API/WebSocket全套功能。企业版额外提供私有部署、团队管理和SSO集成。

可以按月订阅吗？

可以。专业版支持月付($29/月)和年付($299/年)。年付相当于赠送2个月免费使用。

支持哪些编程语言？

Python、JavaScript、TypeScript、Java、Go、Rust、C++。更多语言正在持续添加中。

数据安全如何保障？

所有代码分析在服务端完成，结果仅保留24小时后自动删除。企业版支持本地部署，代码永不离开你的网络。

🔧 方式一：在线使用

1. 访问 API 文档

2. 在 Swagger UI 点击 "Authorize" 输入 API Key

3. 选择代码语言，粘贴代码，点击分析

4. 查看分析结果

🔌 方式二：CLI 命令行

安装 CLI：

pip install codereview-ai

设置 API Key：

export CODEREVIEW_API_KEY=''

分析文件：

codereview analyze myfile.py

🐍 方式三：Python SDK

import requests

resp = requests.post(

json={"code": "print('hello')", "language": "python"}
)
print(resp.json())
